RioSEO Social tracking Code.The code below includes the Tag and Trace component which enables tracking of social sharing by generating an 11 digit unique ID for each user and appends this ID to the URL in the browser environment. In addition, it will also configure the Sharing Tool. Ideally, this code should be placed before the closing </head> tag in your site's template or page.Preferred Hash configuration (appends Meteor ID behind the # in URL):<script type="text/javascript" src="http://static.meteorsolutions.com/metsol.js"></script><script src="http://static.meteorsolutions.com/metshare.js"></script><script type="text/javascript">meteor.tracking.track('c9994adf-8f32-4007-9571-c60e0a3009fe', {'url_storage_source': 'hash', 'cookie_domain': '.laseraway.net', 'query_string_tag_key': ['mtag', 'source', 'src']});meteor.orion.init();meteor.sharing.superbar.init({'id': '1', 'parentID': 'meteorShare', 'size' : 'large', 'orientation' : 'horizontal', 'sites' : ['facebook', 'twitter', 'email' ]});</script>That's it. This will enable tracking of your site and must execute first on the page in order to utilize the rest of technologies listed below.STEP 2. Conversion/End-Action TrackingWe offer the ability to name tracking events so that they are easily distinguished, giving you a great deal of flexibility in how you measure your campaign or site's effectiveness.***In all cases - please rename the 'myconversion' to a conversion name appropriate for a specific end action i.e. 'whitepaper1_download', 'register_page1', 'watch_welcomeVideo'Option One -  Javascript  (The preferred method of tracking conversions) This calls our track_conversion() method:<input type="button" onClick="meteor.tracking.track_conversion('c9994adf-8f32-4007-9571-c60e0a3009fe',{'name':'myconversion'});">_____HREF Javascript Configuration (use for conversion that are links (<a href=...))<a href="http://example.com" onclick="meteor.tracking.track_conversion('c9994adf-8f32-4007-9571-c60e0a3009fe',{'name':'myconversion'}, this);">***The "this" is needed to prevent the original link from firing, which can cause issues. If you are already blocking the original link, use the "non-this" format.Option Two -  PixelFor tracking on third-party sites, or places where incorporating Javascript is not possible, use the pixel conversion:<img src="http://cdnt.meteorsolutions.com/api/conversion_pixel?cookie_name=c9994adf-8f32-4007-9571-c60e0a3009fe&name=myconversion" />STEP 3. Sharing ToolTO USE BUILT-IN SHARETOOL ICONSPlease create a <div id='meteorShare'></div> (the id value may be different but it must be same as the parameter of 'parentID' when meteor.sharing.superbar.init call is made) where the Meteor Share Tool is to be positioned and displayed on the web page.If present on the page, Meteor share tool will automatically use the value of HTML <title> tag to set the title of Facebook sharing and email subject and meta name="description" content will be used for Twitter, Facebook and Linkedin shared messages and the email body text. Optionally, title and/or message can be explicitly set which will override the value of HTML <title> and meta name="description" by adding parameters of 'title' and/or 'message' to meteor.sharing.superbar.init call in the section above. Please see "Superbar Documentation 11_16_12 Release" for further details and configuration options.
