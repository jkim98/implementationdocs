1. Meteor tracking Code.
The code below is the Tag and Trace component which enables tracking of social sharing by generating an 11 digit unique ID for each user and appends this ID to the URL in the browser environment. 
Ideally, this code should be placed at the header section of the every page where Meteor social tracking is desired.
<script type="text/javascript" src="//cdnt.meteorsolutions.com/metsol.js"></script>
<script type="text/javascript">
    meteor.tracking.track('4ceeb6f8-26dc-43f3-9b0a-d6ad1b23a566', {'url_storage_source': 'hash' , 'query_string_tag_key': ['mtag', 'source', 'src', 'cr_cc', 'CR_CC', 'WT.mc_id']});
    meteor.orion.init();
</script>

2. Share Tool Implementation
Please name a DIV tag where you would like the Share Tool to be rendered in the page:  <div id='SET DIV ID'> 
Then insert the following code into the body section of the HTML and replace 'SET DIV ID' with the actual DIV id. 
<script src="//cdnt.meteorsolutions.com/metshare.js"></script><script>
    if (meteor.detect_browser().device === "mobile") {
        meteor.sharing.superbar.init({'id': 'med_mobile' , 'device' : 'mobile', 'size' : 'mobile-small', 'page_position' : 'right', 'sites' : ['twitter', 'facebook', 'email']});
    } else {
        meteor.sharing.superbar.init({'id': 'med_standard' ,'parentID' : 'SET DIV ID', 'size' : 'med', 'orientation' : 'vertical', 'sites' : ['twitter', 'facebook', 'linkedin',  'email']});
    }
</script>
If present on the page, Meteor share tool will automatically use the value of HTML <title> tag to set the title of Facebook sharing and email subject and meta name="description" content will be used for Twitter, Facebook and Linkedin shared messages and the email body text. 
Optionally, title and/or message can be explicitly set which will override the value of HTML <title> and meta name="description" by adding parameters of 'title' and/or 'message' to meteor.sharing.superbar.init call in the section above. 
Please see "Superbar Documentation 11_16_12 Release" for further details and configuration options.      3. Conversion/End-Action TrackingWe offer the ability to name tracking events so that they are easily distinguished, giving you a great deal of flexibility in how you measure your campaign or site's effectiveness.***In all cases - please rename 'myconversion' to a conversion name appropriate for a specific end action i.e. 'whitepaper1_download', 'register_page1', 'watch_welcomeVideo'Option One -  Javascript  (The preferred method of tracking conversions) This calls our track_conversion() method<input type="button" onClick="meteor.tracking.track_conversion('4ceeb6f8-26dc-43f3-9b0a-d6ad1b23a566',{'name':'myconversion'});">_____HREF  Javascript Configuration(use for conversion that are links (<a href=...))<a href="http://example.com" onclick="meteor.tracking.track_conversion('4ceeb6f8-26dc-43f3-9b0a-d6ad1b23a566',{'name':'myconversion'}, this);">***The "this" is needed to prevent the original link from firing, which can cause issues. If you are already blocking the original link, use the "non-this" format.Option Two -  PixelFor tracking on third-party sites, or places where incorporating Javascript is not possible, use the pixel conversion:<img src="//cdnt.meteorsolutions.com/api/conversion_pixel?cookie_name=4ceeb6f8-26dc-43f3-9b0a-d6ad1b23a566&name=myconversion" />
4. Registering User IDTo associate an user id to a user, call meteor.tracking.register_key(user_id) when you have an user_id to associate with the user. For example, this is typically done after registration or login to associate the username/user id (like an email address) to the user.

<script type="text/javascript" src="//cdnt.meteorsolutions.com/metsol.js"></script>
<script type="text/javascript">
var user_id = "john@email.com"meteor.tracking.register_key(user_id);
</script>
